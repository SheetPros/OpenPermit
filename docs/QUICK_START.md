# Quick Start: Get Value in 15 Minutes

## The Quick Win Path
Get immediate value with minimal setup - just JSON-LD + one core standard.

### Step 1: Install CLI (2 minutes)
```bash
npm install -g @openpermit/cli
openpermit init my-permit-system
cd my-permit-system
```
### Step 2: Choose Your Quick Win (1 minute)
```bash
# Option A: Building Permit Tracking
openpermit generate building-permit

# Option B: Business License Management  
openpermit generate business-license

# Option C: Zoning Application Processing
openpermit generate zoning-application
```
### Step 3: Configure Basic Settings (5 minutes)
```bash
openpermit config set municipality "Your City Name"
openpermit config set contact-email "permits@yourcity.gov"
openpermit schema validate
```
### Step 4: Launch Demo (2 minutes)
```bash
openpermit demo start
# Opens http://localhost:3000 with sample data
```
### Step 5: See Immediate Benefits (5 minutes)

✅ Standardized permit data structure
✅ Automatic compliance reporting
✅ Developer-friendly APIs
✅ Public transparency dashboard
